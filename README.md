# Projet-Fi-tia

Simple Vite + React + Tailwind scaffold for the Fi"tia presidential program website.

Quick start

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Deploy to Vercel (web or CLI):

1. From the Vercel dashboard, import the Git repository and set the framework to "Other" or let Vercel detect it.
2. Or use the CLI after installing `vercel`:

```bash
npx vercel login
npx vercel --prod
```

Notes
- Replace `src/assets/candidate.svg` with an official photo (keep the same filename or update `App.jsx`).
- If you need a custom domain, configure it in the Vercel dashboard.
